PROJECT TITLE : Hospital Billing Management System

Project Description :

This is a backend-focused web application built using Flask and Firebase.
The system helps manage hospital billing records with secure user authentication.
The project focuses mainly on backend logic, APIs, authentication, and database operations.
Problem Statement :

Small hospitals often manage billing manually or using spreadsheets, which can lead to errors, data loss, and poor security.
This project provides a centralized and secure billing system with authentication and structured data storage.
Tech Stack :

Backend: Python, Flask
Authentication: Firebase Authentication
Database: Firebase Firestore
Frontend: HTML, Bootstrap (template-based)
Tools: Postman, Git

How to Run Locally :

Clone the repository

Install dependencies: pip install -r requirements.txt

Copy config.example.py and rename it to config.py and replace the actual firebase api key from the firebase consoe

Add your Firebase Web API key in config.py

Run the application: python app.py

IMPORTANT NOTE :

Firebase API keys and service account files are excluded from the repository.

Sensitive files are ignored using .gitignore.

A sample configuration file is provided for setup reference.
